SSH Pilot APT Repository

This is an unofficial APT repository for SSH Pilot, a tool for managing SSH connections.

Official Project: github.com/mfat/sshpilot

Installation

To use this repository, follow these steps:

1. Add the GPG key

curl -fsSL https://mahdi.github.io/sshpilot-ppa/pubkey.gpg | sudo gpg --dearmor -o /usr/share/keyrings/sshpilot-ppa.gpg

2. Add the repository

echo "deb [signed-by=/usr/share/keyrings/sshpilot-ppa.gpg arch=amd64] https://mahdi.github.io/sshpilot-ppa any main" | sudo tee /etc/apt/sources.list.d/sshpilot-ppa.list

3. Update and install

sudo apt update
sudo apt install sshpilot

Updating

Once installed, you can update SSH Pilot along with your other packages:

sudo apt update && sudo apt upgrade

Uninstallation

To remove SSH Pilot and this repository:

# Remove the package
sudo apt remove sshpilot

# Remove the repository
sudo rm /etc/apt/sources.list.d/sshpilot-ppa.list
sudo rm /usr/share/keyrings/sshpilot-ppa.gpg

# Update package lists
sudo apt update
Note: This is an unofficial repository. The packages are automatically fetched from the official SSH Pilot releases and republished here for convenience.

About

This repository is automatically updated when new versions of SSH Pilot are released. The packages are signed with a GPG key for security.
